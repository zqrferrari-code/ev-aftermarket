# Page Routes

Base URL: `http://localhost:3000`

## Static Pages

| Page | Path |
|------|------|
| Home | `/` |
| Privacy Policy | `/privacy` |
| Contact | `/contact` |

## Market Pages

`[market]` — market code, e.g. `us`, `au`, `cn`

| Page | Path |
|------|------|
| Market Home | `/[market]` |
| Model Detail | `/[market]/models/[slug]` |
| Problems Index | `/[market]/problems` |
| Model Problems | `/[market]/problems/[model]` |
| Charging Info | `/[market]/charging/[model]` |
| Service | `/[market]/service/[model]` |
| Dealers | `/[market]/dealers/[brand]/[state]` |
| DTC Code List | `/[market]/dtc/[model]` |
| DTC Code Detail | `/[market]/dtc/[model]/[code]` |
| Software Updates | `/[market]/updates/[model]` |
| Software Update Version | `/[market]/updates/[model]/[version]` |
