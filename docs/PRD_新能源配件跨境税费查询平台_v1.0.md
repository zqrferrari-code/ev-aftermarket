## 一、产品概述

### 1.1 产品背景

中国新能源汽车出口量2025年达343万辆（同比增长70%），但配件跨境出口的配套服务严重滞后：
- 配件HS编码归类混乱，卖家随意申报
- 跨境税费不透明，买家常被要求补税
- 缺乏专门面向新能源配件的查询工具
- 速卖通、亚马逊等平台有商品但无税费指导

### 1.2 产品目标

**短期目标（MVP）**：
- 成为"中国新能源车配件跨境税费查询第一站"
- 覆盖10个热门车型 × 20个高频配件 × 3个核心市场
- 日UV达到200，联盟点击转化率达到5%

**中期目标（6-12个月）**：
- 覆盖50个车型 × 50个配件 × 8个市场
- 推出高级会员订阅服务
- 月活跃用户数达到5000

**长期目标（12-24个月）**：
- 成为配件跨境出口的信息基础设施
- 推出B端API数据服务
- 探索配件撮合交易模式

### 1.3 一句话卖点

> 为中国新能源汽车配件跨境出口提供"编码-税费-供应商"一站式信息查询服务

### 1.4 名词解释

| 术语 | 解释 |
|------|------|
| **HS编码** | 海关编码，全球通用的商品分类编码系统，前6位国际统一 |
| **CIF** | 到岸价格 = FOB货值 + 运费 + 保险费 |
| **DDP** | 完税后交货，卖家承担所有费用和风险 |
| **OEM号** | 原厂配件编号 |
| **UN38.3** | 锂电池运输安全测试标准 |
| **CE认证** | 欧盟强制性产品认证 |
| **E-mark** | 欧洲经济委员会车辆零部件认证 |
| **SASO** | 沙特阿拉伯标准组织认证 |
| **联盟链接** | 带有推广追踪ID的电商链接，成交后返佣 |
| **反补贴税** | 对接受出口补贴的商品征收的附加税 |

## 二、目标用户与用户故事

### 2.1 用户角色

#### 角色A：海外个人车主（C端）

| 属性 | 描述 |
|------|------|
| **人口特征** | 25-45岁，居住在欧洲/东南亚/中东，已购买中国新能源车 |
| **核心痛点** | 车辆损坏后不知如何购买配件、担心被税、语言不通 |
| **使用场景** | 保险杠撞坏 → 搜索配件 → 查询税费 → 决定是否购买 |
| **付费意愿** | 中（愿为确定性付费，但价格敏感） |
| **代表用户** | 德国比亚迪Atto 3车主、泰国哪吒V车主 |

#### 角色B：海外独立维修厂（B端-small）

| 属性 | 描述 |
|------|------|
| **人口特征** | 小型维修厂老板/技师，1-5人团队 |
| **核心痛点** | 没修过中国电车，不敢接单；买不到配件；不懂编码 |
| **使用场景** | 客户送来中国电车 → 查询配件编码和税费 → 报价给客户 |
| **付费意愿** | 高（关系到生意存亡，愿为效率工具付费） |
| **代表用户** | 波兰华沙电动车专修厂、迪拜汽车维修店 |

#### 角色C：国内跨境电商卖家（B端-small）

| 属性 | 描述 |
|------|------|
| **人口特征** | 在速卖通/亚马逊销售汽配的个人/小团队卖家 |
| **核心痛点** | 不确定配件编码是否正确、不知道如何向买家解释税费 |
| **使用场景** | 上架新品前 → 查询正确编码 → 确认申报要素 → 填写商品信息 |
| **付费意愿** | 中（愿为合规和效率付费） |
| **代表用户** | 广州陈田拆车件卖家、浙江台州汽配出口商 |

#### 角色D：海外保险公司定损员（B端）

| 属性 | 描述 |
|------|------|
| **人口特征** | 保险公司理赔部门员工 |
| **核心痛点** | 不懂中国新能源车配件价格，定损靠猜 |
| **使用场景** | 接到中国电车理赔 → 查询配件价格和税费 → 出具定损报告 |
| **付费意愿** | 高（影响理赔效率，公司报销） |
| **代表用户** | 德国HUK-Coburg保险、泰国Viriyah保险 |

### 2.2 用户故事

#### US-001：海外车主查询配件税费

> **作为** 德国比亚迪海豚车主
> **我想** 查询前保险杠出口到德国的税费
> **以便** 知道在速卖通购买时实际要付多少钱

**验收标准**：
- 输入"比亚迪 海豚 前保险杠 德国"后3秒内显示结果
- 显示HS编码、关税率、增值税率、预计总成本
- 提供速卖通同款配件链接和价格对比

#### US-002：维修厂批量查询

> **作为** 波兰维修厂老板
> **我想** 一次性查询多个配件的编码和税费
> **以便** 给客户出具准确的维修报价单

**验收标准**：
- 支持批量输入（Excel粘贴或表格填写）
- 支持导出PDF/Excel报告
- 显示每个配件的CIF价格和税费明细

#### US-003：卖家确认编码合规

> **作为** 速卖通汽配卖家
> **我想** 确认我的保险杠产品应该用什么HS编码
> **以便** 避免海关查验和买家纠纷

**验收标准**：
- 输入配件描述后推荐最可能的编码
- 显示申报要素（材质、功能、电压等）
- 标注常见归类误区

#### US-004：保险公司定损参考

> **作为** 保险公司定损员
> **我想** 查询中国电动车配件的海外维修成本
> **以便** 合理确定理赔金额

**验收标准**：
- 显示配件价格区间（速卖通/国际站/1688）
- 显示税费和物流成本
- 显示维修工时参考


**字段说明**：

| 字段 | 类型 | 必填 | 规则 |
|------|------|------|------|
| FOB货值 | 数字 | 是 | >0，默认展示速卖通参考价 |
| 运费 | 数字 | 是 | ≥0，默认根据目标国估算 |
| 保险费率 | 数字 | 否 | 0-10%，默认2% |
| 目标国 | 选择 | 是 | 下拉选择，默认上次选择 |

**计算公式**：
```
CIF = FOB货值 + 运费 + (FOB货值 + 运费) × 保险费率
关税 = CIF × 关税税率
增值税 = (CIF + 关税) × 增值税率
总成本 = CIF + 关税 + 增值税
```

#### 3.2.4 速卖通联盟导流（F-004）

**功能描述**：在查询结果页展示相关配件的速卖通商品卡片

**数据来源**：
- 速卖通联盟API（affiliate.aliexpress.com）
- 搜索关键词：配件英文名称 + 车型英文名称

**展示规则**：
- 最多展示3个商品卡片
- 按价格从低到高排序
- 显示：商品图、价格、评分、销量、平台标识
- 点击后新开标签页跳转速卖通

**联盟链接格式**：
```
https://s.click.aliexpress.com/e/[tracking_code]
?dp=[product_id]
&af=[affiliate_id]
```

**fallback机制**：
- API失效时显示"暂无供应信息"
- 提供手动搜索链接（速卖通搜索页）
