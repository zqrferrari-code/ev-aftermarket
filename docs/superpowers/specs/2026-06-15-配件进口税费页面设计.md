# 设计文档：EV 配件进口税费页面（澳洲市场）

**日期**：2026-06-15
**状态**：草稿——待用户确认
**范围**：MVP——比亚迪配件，仅澳洲市场

---

## 一、问题与机会

EVAftermarket.io 目前没有任何配件相关内容。澳洲中国新能源车主在需要更换外观件时，不清楚从速卖通或国内供应商购买配件需要缴纳多少进口税费，这种信息不透明导致购买顾虑和放弃下单。

机会点：目前几乎没有任何英文内容覆盖"BYD Dolphin front bumper import duty Australia"这类搜索词。这类长尾 SEO 页面竞争极少，可以快速排名并带动联盟点击。

**现有系统流量尚低**，因此策略是建立能独立吸引搜索流量的页面，而不是依靠现有 DTC 页面带流量。

- 用户账号、查询历史、批量查询
- 后台管理系统（数据直接写入数据库）

## 三、URL 结构

沿用现有系统的路由风格：

```
/au/parts/[brand]/[model]/[part-slug]
```

示例：
```
/au/parts/byd/dolphin/front-bumper
/au/parts/byd/dolphin/headlights
/au/parts/byd/atto-3/rear-bumper
/au/parts/byd/seal/side-mirror
```

索引页：
```
/au/parts                     → 所有品牌（MVP 只有比亚迪）
/au/parts/byd                 → 比亚迪车型列表
/au/parts/byd/dolphin         → 海豚配件列表
```

HS 编码详情页：
```
/au/parts/hs/[code]           → HS 编码详情
```

示例：
```
/au/parts/hs/8708.10          → 保险杠编码详情
/au/parts/hs/8512.20          → 大灯/尾灯编码详情
/au/parts/hs/8708.29          → 翼子板等车身件编码详情
/au/parts/hs/7009.10          → 后视镜编码详情
```

配件详情页与 HS 编码详情页互相内链：
- 配件页的"HS 编码"字段链接到对应编码详情页
- 编码详情页列出所有使用该编码的配件并回链

### 5.2 MVP 涉及的 HS 编码

| 配件 | 澳洲进口编码（8位） | 中国出口编码（10位）| MFN 关税 | GST |
|------|-----------------|-----------------|---------|-----|
| 前/后保险杠 | `8708.10.00` | `8708101000` | 0% | 10% |
| 大灯总成 | `8512.20.00` | `8512201000` | 0% | 10% |
| 尾灯总成 | `8512.20.00` | `8512209000` | 0% | 10% |
| 前翼子板 | `8708.29.10` | `8708299090` | 0% | 10% |
| 后视镜 | `8708.29.20` | `8708291000` | 0% | 10% |

**注：** 澳洲对上述所有配件的 MFN 关税已为 0%，与原产地无关，无需原产地证书。ChAFTA 在此场景下无额外优惠（已无可降）。

### 4.2 计算公式

```
CIF = 配件价格 + 运费
关税 = CIF × 关税税率   // ChAFTA 0%，MFN 备选 5%
GST = (CIF + 关税) × 10%
总成本 = CIF + 关税 + GST
```

用户可在 ChAFTA（0%）和 MFN（5%）两种场景之间切换。

## 六、数据模型

新建四张表，添加至现有 Supabase PostgreSQL 实例。设计原则：**编码和税率均按国家存储**，新增市场只需插入数据行，不改表结构。

### 配件表 `parts`

不存任何特定国家的编码，只存配件本身的属性。

```sql
CREATE TABLE parts (
  id           SERIAL PRIMARY KEY,
  slug         VARCHAR(100) NOT NULL UNIQUE,
  name_en      VARCHAR(100) NOT NULL,
  name_cn      VARCHAR(100),
  category     VARCHAR(50),    -- 'exterior'、'lighting' 等
  material     VARCHAR(100),
  is_dangerous BOOLEAN DEFAULT false,
  notes        TEXT,
  created_at   TIMESTAMP DEFAULT NOW()
);
```

### 配件车型适配表 `part_model_compatibility`

```sql
CREATE TABLE part_model_compatibility (
  id         SERIAL PRIMARY KEY,
  part_id    INT REFERENCES parts(id),
  model_id   INT REFERENCES models(id),
  years      VARCHAR(50),    -- 如 '2022-2025'
  oem_number VARCHAR(100),
  notes      TEXT
);
```

### 各国 HS 编码表 `part_hs_codes`

每个配件在不同国家的海关编码单独一行存储。

```sql
CREATE TABLE part_hs_codes (
  id               SERIAL PRIMARY KEY,
  part_id          INT NOT NULL REFERENCES parts(id),
  country_code     CHAR(2) NOT NULL,  -- ISO 3166-1: 'CN'、'AU'、'DE'、'IQ' 等
  hs_code          VARCHAR(12) NOT NULL,
  -- CN: 10位出口编码（含监管条件）；AU/DE/IQ: 8位进口编码
  hs_code_type     VARCHAR(10) NOT NULL, -- 'export'（出口）或 'import'（进口）
  description_en   TEXT,               -- 法定品名（英文）
  description_local TEXT,              -- 法定品名（当地语言）
  declaration_elements TEXT,           -- 申报要素
  regulatory_conditions VARCHAR(100),  -- 监管条件（主要用于 CN 出口编码）
  last_verified    DATE,
  source_url       VARCHAR(255),
  notes            TEXT,
  UNIQUE (part_id, country_code, hs_code_type)
);
```

示例数据（前保险杠）：

| part_id | country_code | hs_code | hs_code_type | description_en |
|---------|-------------|---------|-------------|---------------|
| 1 | CN | 8708101000 | export | 乘用车保险杠 |
| 1 | AU | 87081010 | import | Bumpers and parts thereof |
| 1 | DE | 87081010 | import | Stoßstangen und Teile davon |

### 关税税率表 `tariff_rates`

存储从中国出口到各目标国的进口税率，支持多种优惠税率机制。

```sql
CREATE TABLE tariff_rates (
  id               SERIAL PRIMARY KEY,
  country_code     CHAR(2) NOT NULL,   -- 进口国，如 'AU'、'DE'、'IQ'
  hs_code          VARCHAR(12) NOT NULL, -- 该进口国的 HS 编码
  mfn_rate         DECIMAL(5,2),       -- 最惠国税率（%）
  fta_name         VARCHAR(50),        -- 自贸协定名称，如 'ChAFTA'、'RCEP'
  fta_rate         DECIMAL(5,2),       -- 自贸协定优惠税率（%）
  fta_conditions   TEXT,               -- 享受优惠税率的条件（如原产地证书要求）
  vat_rate         DECIMAL(5,2),       -- 增值税 / GST（%）
  additional_duties TEXT,              -- 附加税说明（如反倾销税、反补贴税）
  last_verified    DATE,
  source_url       VARCHAR(255),
  UNIQUE (country_code, hs_code)
);
```

示例数据（保险杠，HS 8708.10）：

| country_code | hs_code | mfn_rate | fta_name | fta_rate | fta_conditions | vat_rate |
|-------------|---------|---------|---------|---------|--------------|---------|
| AU | 87081010 | 5.00 | ChAFTA | 0.00 | 需提供原产地证书（CCPIT/CCIB签发） | 10.00 |
| DE | 87081010 | 3.50 | — | — | — | 19.00 |
| IQ | 87081010 | 5.00 | — | — | — | 0.00 |

**MVP 只填 AU 的数据**，表结构已为其他国家预留位置。

## 七、速卖通联盟集成

### 接入方式
- 注册速卖通联盟账号（affiliate.aliexpress.com）
- 使用商品搜索 API，按关键词获取商品
- 查询结果缓存至 Supabase，有效期 24 小时，避免触发频率限制
- 每个配件页最多展示 2-3 张商品卡片

### 搜索关键词策略
```
"{配件英文名} {车型英文名} {年款}"
例：front bumper BYD Dolphin 2022
```

### 降级方案
API 不可用或无返回时，展示静态"在速卖通搜索"链接（预填关键词），不出现页面错误。

### 合规标注
- 商品卡片区域明确标注"推广链接 / Affiliate"
- 与现有站点的免责声明风格保持一致

## 八、SEO 策略

每个页面针对一组特定长尾关键词：

| 页面 | 核心关键词 |
|------|-----------|
| /au/parts/byd/dolphin/front-bumper | "BYD Dolphin front bumper Australia" |
| /au/parts/byd/dolphin/headlights | "BYD Dolphin headlight replacement Australia" |
| /au/parts/byd/atto-3/front-bumper | "BYD Atto 3 front bumper import duty" |
| /au/parts/hs/8708.10 | "HS code 8708.10 Australia duty rate" |
| /au/parts/hs/8512.20 | "8512.20 headlight import duty Australia" |

### 页面 SEO 要素
- H1 包含：配件名称 + 车型 + "import duty" + "Australia"
- Meta description 提及 ChAFTA 0% 关税、GST、速卖通
- Schema.org：每个配件页添加 `Product` 结构化数据
- 面包屑 Schema
- 内链：配件页链接到对应编码详情页；编码页列出所有关联配件并回链；两者均链接到对应车型的 DTC 索引页

### 静态生成
全部约 22-23 个页面通过 `generateStaticParams` 在构建时预渲染，与现有站点方式一致。

## 九、不做的内容

单人开发，约 3 周完成，以下内容不在本次范围内：

- **不做搜索功能**：用户通过 Google 或目录页浏览找到配件

- **不做用户账号**：所有页面匿名访问
- **不做批量查询**
- **不做其他市场**：仅澳洲
- **不做其他品牌**：仅比亚迪
- **不做复杂配件**：仅外观件和灯具

## 十一、实施顺序

1. Supabase 中新建 `parts`、`part_model_compatibility`、`part_hs_codes`、`tariff_rates` 四张表
